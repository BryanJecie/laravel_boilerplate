# Laravel 10 Boilerplate


```bash
# Instructions

# Clone via ssh
$ git clone git@github.com:BryanJecie/laravel_boilerplate.git

# Install dependencies
$ composer install

# Config database
$ cp .env.example .env

# Run commands
$ php artisan key:generate

# Run the Users Database for the first time
$ php artisan migrate:migrate --seed

# Run NPM commands
$ npm install && npm run dev

# Run app via local
$ php artisan serve
```
